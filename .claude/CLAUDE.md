# GAN - Non-profit for Soldiers and Families

## Project
Non-profit organization (amuta) supporting soldiers and their families.
Current focus: Building organization website.

## Domain Focus
- Website development -> use code-explorer agent
- Content and design -> use content-creator agent
- Frontend UI -> use content-creator agent with frontend-design skill

## Languages
- Primary: Hebrew (RTL)
- Secondary: Russian
- All content bilingual

## Key Files
- Logo: `logo/`
- Website structure: `questionnaire-and-structure/`
- Content questionnaire: `questionnaire-for-website-content.txt`

## Brand Guidelines
- Tone: warm, supportive, professional
- Sensitive content: no soldier location/unit details, blur faces without permission
- Donations: link to external platforms (Bit, bank transfer)

## Target Audiences
- Donors (private and institutional)
- Soldiers and families needing support
- Volunteers
- Government/authorities
